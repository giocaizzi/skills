# Agents Skills

This is my personal set of skills for AI Agents. 

## Specification

Read the specification https://agentskills.io/home

## Usage

```bash
# clone the repository
git clone giocaizzi/skills
# navigate to the repository
cd skills
# copy the specific skill folder you want to use into your agent's `skills` folder
cp -r skills/<skill_name> /path/to/your/agent/skills/
```
